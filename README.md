# EasyUI
一个原生UI控件拓展，快速开发应用，告别重复造轮子。

如何使用
----
在Module build.gradle 添加以下依赖

    implementation 'com.github.shuqingcc:EasyUI:0.0.4'
    or
    implementation 'com.qinssen.easyui:EasyUI_Lib:0.0.4'
    
在项目根目录加入

    allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
